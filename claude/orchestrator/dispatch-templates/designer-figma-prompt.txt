## 输出要求

### 提示词结构

每批提示词包含:
1. **Style Prefix**: 设计风格/主色调/字体/圆角
2. **Web Responsive Design Rules**: 响应式断点规范
3. **App Context**: 应用名/角色/强调色
4. **Pages**: 每个页面 Mobile (375x812) + Desktop (1440x900)

### 批次拆分规则

- 每批提示词总长度 ≤ 5000 字符
- 按页面逻辑分组，每批 2-4 个页面
- 编号从 01 开始

### 每页必须包含

- 布局描述、组件列表、状态说明
- 默认态 / 加载态 / 空态 / 错误态
- mobile 和 desktop 需要独立描述，不是简单放大
